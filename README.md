# GNB

Gloiath National Bank

## Migrations

Add Migrations: `dotnet ef migrations add <MigrationName> --startup-project ./GNB.Api/ --project ./GNB.Data/`

Update Database: `dotnet ef database update --project ./GNB.Api/`
